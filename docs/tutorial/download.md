## 主题使用示例

```jsx | pure

<!-- 引入 defaultTheme 主题 -->
import defaultTheme from '@kdcloudjs/kd-charts/theme/echarts-theme-default.js'


//注册主题
echarts.registerTheme('defaultTheme', defaultTheme);


// 第二个参数可以指定前面引入的主题
var chart = echarts.init(document.getElementById('main'), 'defaultTheme');
chart.setOption({
    ...
});

```
